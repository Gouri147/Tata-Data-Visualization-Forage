#  Project Documentation

## Data Source
Online Retail Dataset (UCI Repository)

## Data Preparation
- Removed invalid records
- Filtered negative values
- Created revenue metric

## Analysis Approach
- Time series analysis for revenue trends
- Country-level performance evaluation
- Product-level revenue analysis
- Customer segmentation based on frequency

## Key Insights
- Revenue is seasonal
- Few countries dominate sales
- Few products drive most revenue
- Customer concentration is high

## Business Impact
- Helps identify expansion opportunities
- Improves marketing targeting
- Supports inventory planning
